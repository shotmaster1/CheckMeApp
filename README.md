# CheckMe – Mental Health Tracker for Students

## Overview
CheckMe is a Python-based application designed to help college students track their mental health through daily check-ins. The app collects mood and stress data, provides feedback, and helps users identify patterns over time.

## Features Planned
- Daily mental health checklist
- Mood and stress tracking
- Weekly summary dashboard
- Smart feedback system
- Action suggestions

## Tech Stack
- Python
- Tkinter
- SQLite
- Pandas
- Matplotlib
